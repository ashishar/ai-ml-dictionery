# AI & ML Dictionary

A comprehensive, interactive, and high-fidelity reference guide designed specifically for **students and AI/ML enthusiasts** who are interested in this emerging field.

This application provides clear, rigorous definitions for over 50 fundamental and advanced concepts in Artificial Intelligence and Machine Learning, ranging from foundational statistics to state-of-the-art transformer architectures.

## 🚀 Key Features

- **Interactive Explorer**: Navigate through categories like Neural Networks, Generative AI, NLP, and Optimization.
- **Academic-Grade Math**: All technical formulations are rendered using professional LaTeX notation (via KaTeX) to ensure precision and clarity.
- **Technical Diagrams**: Integrated visual aids to demystify complex architectures like GANs, Transformers, and Autoencoders.
- **Curated Resources**: Each term is accompanied by direct links to SOTA papers, GitHub repositories, video tutorials, and technical documentation.
- **Open Knowledge**: Content is synthesized from Wikipedia and other trusted academic repositories under the CC BY-SA 4.0 license.

## 🛠️ Technology Stack

- **Framework**: React 18 + Vite
- **Styling**: Vanilla CSS with modern Glassmorphism aesthetics
- **Math Rendering**: React-KaTeX
- **Icons**: Lucide-React
- **Deployment**: GitHub Actions + GitHub Pages

## 📖 Usage

This dictionary is built to serve as a desktop companion for researchers and students. Simply search for a term or browse by category to access deep-dive technical discussions and supporting resources.
